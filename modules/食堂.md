# 食堂

## 食堂分布

北林校园内有多个食堂，分布在不同区域，满足不同口味和需求。

| 食堂名称 | 位置 | 营业时间（参考） | 特色 |
|----------|------|-----------------|------|
| 第一食堂（学生食堂）| 北区宿舍附近 | 6:30–20:30 | 主食、炒菜，价格实惠 |
| 第二食堂 | 教学区附近 | 7:00–20:00 | 多档口，选择丰富 |
| 清真食堂 | 校园内 | 7:00–20:00 | 清真食品，穆斯林同学首选 |
| 教工餐厅 | 主楼附近 | 11:00–14:00 | 学生也可就餐，品质稍高 |
| 风味餐厅 | 部分食堂二楼 | 11:00–14:00 | 特色菜品，档次稍高 |
| 树人公寓餐厅 | 树人公寓内 | 7:00–20:00 | 方便公寓住户 |

> **注意**：以上营业时间为大致参考，寒暑假及节假日期间会有调整，以食堂公告为准。

## 消费方式

- **校园一卡通**：刷卡消费，可在校内充值机或网上充值
- **支付宝/微信**：部分档口支持扫码支付
- **现金**：一般不接受现金，建议使用一卡通

## 价格参考

| 类型 | 价格区间 |
|------|----------|
| 主食（米饭、馒头等）| 0.5–2 元 |
| 普通炒菜 | 3–8 元 |
| 套餐 | 8–15 元 |
| 特色窗口 | 10–20 元 |

## 热门窗口推荐

- **麻辣烫/冒菜**：深受学生喜爱，可自选食材
- **盖浇饭**：分量足，性价比高
- **包子/饺子**：早餐首选
- **煎饼果子**：早餐摊，排队需早
- **沙县小吃风格档口**：快手、实惠

## 实用技巧

1. 早餐高峰期（7:30–8:30）建议早去或错开时间
2. 午餐高峰期（11:30–12:30）排队较长，可稍早或稍晚就餐
3. 一卡通余额不足可在食堂自助充值机补充
4. 天气热时尽量不要在宿舍存放隔夜食物
5. 部分食堂设有餐具消毒区，可自行清洁
