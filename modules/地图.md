# 地图

## 校园概览

北京林业大学位于北京市海淀区清华东路 35 号，紧邻清华大学、圆明园，地处北京高校聚集的学院路区域。

```
地址：北京市海淀区清华东路 35 号
邮编：100083
主校门（南门）：面向清华东路
北门：通往圆明园方向
```

## 重要地点导览

| 地点 | 位置说明 |
|------|----------|
| 图书馆 | 校园中心区域，标志性建筑 |
| 主楼（行政楼） | 校园中轴线北侧 |
| 学研中心 | 南门附近，含研究生宿舍 |
| 体育馆 | 校园西侧 |
| 田径场（操场） | 体育馆旁 |
| 医务室（校医院）| 校园南区，近体育场 |
| 超市（学生超市）| 宿舍区附近，多处分布 |
| ATM | 图书馆旁、食堂附近 |
| 快递驿站 | 宿舍楼附近，各区均有 |

## 在线地图

- **高德地图**：搜索「北京林业大学」即可导航至校门
- **百度地图**：搜索「北京林业大学」，可查看卫星图
- **微信小程序**：搜索「北林校园导航」（第三方，仅供参考）

## 周边交通

| 交通方式 | 路线 |
|----------|------|
| 地铁 | 最近地铁站：13 号线「五道口」（步行约 20 分钟）或乘公交中转 |
| 公交 | 校门口有多条公交线路：331、375、432、562、运通 105 等 |
| 共享单车 | 校内及周边均有哈啰、美团、青桔投放 |
